============================================
  Portable WBS Tool - 簡易マニュアル
============================================

【使い方】

1. index.html をダブルクリックして開く
   → ブラウザが起動し、すぐに使えます

2. 初回起動時はサンプルデータが表示されます

3. 「+ 新規タスク」ボタンでタスクを追加

4. タスク行をクリックで選択、ダブルクリックで編集

--------------------------------------------

【主な機能】

■ タスク管理
  - タスクの追加・編集・削除
  - 親子関係（階層構造）
  - 担当者割り当て
  - 進捗率管理
  - マイルストーン設定

■ ガントチャート
  - 日付グリッドでのタスク表示
  - 今日の日付がハイライト
  - 週末グレー表示

■ ユーザー管理
  - 担当者の追加・削除

■ データ管理
  - 自動保存（ブラウザのLocalStorage）
  - JSONエクスポート/インポート

--------------------------------------------

【注意事項】

・データはブラウザに保存されます
・別のブラウザでは別のデータになります
・定期的にエクスポートでバックアップを推奨

--------------------------------------------

【トラブルシューティング】

Q: データが消えた
A: ブラウザのキャッシュ削除やプライベートモードで
   消える場合があります。エクスポートでバックアップしてください。

Q: 別のPCで使いたい
A: エクスポートしたJSONファイルを持っていき、
   インポートしてください。
